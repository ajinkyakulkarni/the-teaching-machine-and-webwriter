Use cases for self-checking quizzes

Fill in the expression

Actors: The student, the RTM server

Main Flow

Precondition: The quiz question is in its initial state. An RTM is connected.

  1. The student reads the question text and fills the expression text box.
  2. Once the expression text is valid, the system enables the start button.
  3. The student clicks on the start button.
  4. The system reveals an execution panel (see diagram) with some initial values for the input variables already filled in.
